# GTM Week 2 — Demo + Proposal Engine

هذا الملف ينفّذ To-do الأسبوع 2: تثبيت demo تنفيذي موحّد وإرسال 4+ عروض Pilot.

## الهدف الأسبوعي (Gate)

- Demo script موحد (20 دقيقة) يستخدم نفس التسلسل كل مرة.
- 4 عروض Pilot مرسلة على الأقل.
- زمن إرسال العرض ≤ 24 ساعة من نهاية demo.

## تسلسل Demo موحّد

1. Executive Room snapshot (`changes`, `decisions`, `risks`).
2. Approval Center (Class B + HITL).
3. Evidence Pack (source/provenance/verification).
4. سؤال أثر مالي/زمني مع العميل.

## قالب تأهيل العرض

| account | decision_owner | pain_score_1_5 | urgency_1_5 | pilot_fit | proposal_deadline |
|---------|----------------|----------------|-------------|-----------|-------------------|
| | | | | yes/no | YYYY-MM-DD |

## قواعد العرض

- Scope pilot ثابت: 14 يوم، مسار واحد، بدون توسعة features.
- كل عرض يحتوي KPI نجاح واضحة (approval time / cycle time / clarity).
- كل عرض يتضمن next step وتاريخ قرار.

## Exit Criteria

- `proposals_sent >= 4`
- `proposal_turnaround_hours <= 24` للـ demos المؤهلة.
- وجود سجل objections موحّد لدعم الأسبوع 3.
