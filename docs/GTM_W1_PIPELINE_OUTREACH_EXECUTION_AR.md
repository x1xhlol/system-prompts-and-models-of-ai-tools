# GTM Week 1 — Pipeline + Outreach Execution

هذا الملف ينفّذ To-do الأسبوع 1: تشغيل Pipeline + Outreach يومي والوصول إلى 8+ demos محجوزة.

## الهدف الأسبوعي (Gate)

- 60 حسابًا مستهدفًا في القائمة.
- 8 اجتماعات Demo محجوزة (حد أدنى).
- Funnel محدث يوميًا: `Lead > Replied > DemoBooked > Proposal`.

## المهام اليومية

| اليوم | المهمة | المخرج الإلزامي |
|------|--------|-----------------|
| Day 1 | اعتماد ICP + بناء القائمة الأساسية | 60 حسابًا مع مالك/قناة |
| Day 2 | إرسال batch أول | 25 LinkedIn + 20 WhatsApp + 10 Email |
| Day 3 | متابعة batch 1 + batch 2 | تحديث Funnel + حجز اجتماعات |
| Day 4 | متابعة مركزة للمهتمين | 4+ demos مؤكدة |
| Day 5 | إغلاق أسبوع 1 | 8+ demos + قائمة أولويات للأسبوع 2 |

|---------|-------|------------|---------|--------------|----------|--------------|
| YYYY-MM-DD | | | | | | |

## قواعد التنفيذ

- لا فتح segment جديد قبل اكتمال follow-up للsegment الحالي.
- أي lead يرد إيجابيًا يجب تحويله إلى موعد خلال 24 ساعة.
- أي اعتراض متكرر يوثّق في قائمة objections للأسبوع 2.

## Exit Criteria

- `demos_booked >= 8`
- وجود جدول مواعيد فعلي للأسبوع 2.
- عدم وجود leads بلا follow-up أكثر من 48 ساعة.
