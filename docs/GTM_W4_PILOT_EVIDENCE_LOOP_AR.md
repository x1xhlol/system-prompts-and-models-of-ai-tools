# GTM Week 4 — Pilot Delivery + Evidence to Revenue Loop

هذا الملف ينفّذ To-do الأسبوع 4: تشغيل pilot حي وجمع Evidence أثر وربطها بـ case-study + referral.

## الهدف الأسبوعي (Gate)

- pilot حي محدود النطاق يعمل فعليًا.
- 3 أدلة أثر على الأقل (زمن/وضوح/دورة).
- case-study أولية + referral request.

## جدول قياس الأثر

| metric | baseline | week4_value | delta | source |
|--------|----------|-------------|-------|--------|
| approval_time | | | | system log |
| cycle_time | | | | workflow/event |
| decision_clarity | | | | executive review |

## قالب Case Study (صفحة واحدة)

1. المشكلة قبل pilot.
2. ماذا تم تركيبه (scope محدد).
3. ماذا تحسن (الأرقام أعلاه).
4. الخطوة التالية (توسعة أو تجديد).

## Referral Loop

- طلب إحالة رسمي بعد جلسة الأثر.
- تسجيل الإحالة في funnel كـ lead جديد.
- متابعة خلال 72 ساعة.

## Exit Criteria

- pilot status = live.
- case-study draft مكتملة.
- referral واحد على الأقل موثق.
